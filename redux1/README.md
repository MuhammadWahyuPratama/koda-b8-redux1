# Google Forms Survey Clone

A simple survey application inspired by Google Forms built with React. This project uses React Hook Form and Yup for form validation, while submitted responses are stored in Local Storage and displayed on a response page.

## Tech Stack

1. React
2. Vite
3. Tailwind CSS
4. React Hook Form
5. Yup
6. React Router DOM
7. React Icons
